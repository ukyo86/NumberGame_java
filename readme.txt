1. 開発したソフトウェア
[数当てゲーム]:CUI

概要
フジテレビで放送されていた「Numer0n:ヌメロン」を参考に、自分たちで考えたアイテム機能を追加して開発した。
1から9までの任意の数字を3つ選び、相手が選んだ数字3桁全てを当てる推測ゲーム
------------------------------------------------------------------------
<ゲーム内用語>
Hit  :数字と位置があっている
Blow :数字は含まれているが位置が異なる

<ルール>
二人対戦用の数当てゲーム
各自1～9の3桁の数字を設定しHitとBlowのヒントを元に相手の数を当てれば勝利
アイテムは2HItとなったとき、使用可能となる。

例)設定数字: 519 
解答 523 -> 1Hit 0Blow
解答 248 -> 0Hit 0Blow
解答 915 -> 1Hit 2Blow
解答 519 -> 3Hit 0Blow

<アイテム説明>
・Double:2回連続で自分のターンとなる
・HighAndLow:相手の設定数字がHigh(1～4)かLow(5～9)か知ることが出来る。　例)　489　->　LHH
・Shuffle:自分の設定数字をランダムでシャッフルできる
・GetOneNumber:相手の設定数字のうち１つだけ位置と数字を知ることが出来る。 例)　539　-> **9
--------------------------------------------------------------------------

以下4つのファイルから構成
・Main.java            (実行)
・GameController.java  (ゲーム全体の制御:以下２つを制御)
・GameFunction.java    (数当てゲーム機能)
・SocketConnector.java (通信機能)

4.プログラムについて

完成したそれぞれのプログラムについて簡単に述べる。またMain.javaは実行のみであるため省略する。

4.1 GameFunction.javaについて
・順番制御:      誰のターンか、ターンの切り替え
・hit,blow判定: 相手の解答からhit部,blow部に分類し表示
・勝利判定:      相手の解答が自分の解答と一致するかの判定
・アイテム機能:    コマンドの入力でアイテムの使用を可能にする

4.2 SocketConnector.javaについて
・サーバー、クライアント制御: 接続設定,終了の制御を行う
・データの送受信:       順番設定,三桁設定,解答,勝利判定の相互の送受信をまとめて行う

4.3 GameController.javaについて
・通信の設定:  サーバー,クライアントの任意設定、相互の送受信管理
・ゲームの設定: 3桁の設定,順番決め,ターン制による制御
